# Java Core & OOP Fundamentals ☕

Welcome to my Java development portfolio. This repository contains a series of practical exercises, real-world simulations, and capstone projects designed to build a solid foundation in software architecture, Object-Oriented Programming (OOP) principles, and core data structures.

## 🎯 Core Concepts Covered

This repository is structured to demonstrate mastery over the fundamental pillars of Java programming:

### Object-Oriented Programming (OOP)
* **Classes and Objects:** Blueprint creation and memory instantiation.
* **Encapsulation:** Data hiding and access control using `private` modifiers, getters, and setters.
* **Aggregation and Composition:** Designing relationships between objects (Weak and Strong coupling).
* **Inheritance:** Code reusability and hierarchical structures using the `extends` keyword.
* **Polymorphism:** Method overriding (`@Override`) and dynamic method dispatch.
* **Abstract Classes & Interfaces:** Defining strict contracts and unbreakable architectural molds.
* **Exception Handling:** Gracefully managing runtime errors, system failures, and custom business rules using `try-catch-finally` and `throw/throws`.

### Data Structures & Collections Framework
* **Lists (`ArrayList`):** Implementation of dynamic arrays for ordered data storage and custom object filtering.
* **Sets (`HashSet`):** Utilizing Hash-based structures to ensure data uniqueness and high-performance search (No duplicates allowed).
* **Maps (`HashMap`):** Efficient key-value pair management and dictionary architecture for rapid data retrieval.

### Persistence & File Management
* **File I/O:** Reading, writing, and updating physical files (`.txt`, `.csv`) on the hard drive using `FileReader`, `FileWriter`, and `BufferedReader`.
* **Data State Management:** Converting objects from RAM to physical disk storage and vice versa.
